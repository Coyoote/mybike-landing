# MyBike landing page
  Landing page for MyBike
    - [DEMO LINK](https://Coyoote.github.io/mybike-landing)
## Local development
  - Tech stack: HTML5, Sass(SCSS), BEM
  - Responsive design
## Installing 
  1. **Fork** and **Clone** this repository
  3. Run `npm install` (or just `npm i`) in your terminal
  4. Run `npm start`
